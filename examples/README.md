# 示例输出文件

本目录包含工具生成的示例输出文件，帮助用户了解预期结果。

## 文件说明

### 图表文件

#### 市场比值综合分析_示例.png
包含所有5个比值的综合图表：
- 金银比
- 金铂比
- 油金比
- 纳指/标普比
- 道指/黄金比

每个子图包含：
- 比值曲线（彩色线条）
- 历史均值（红色虚线）
- ±1标准差区间（橙色阴影）
- 当前状态统计框

#### 金银比_示例.png
金银比的详细分析图表，包含：
- 5年历史走势
- 均值和标准差区间
- 最高/最低值标注
- 详细统计信息

### 数据文件

#### 市场比值汇总_示例.csv
汇总表格，包含所有比值的：
- 当前值
- 历史均值
- 标准差
- 分位数
- 状态判断（偏高/正常/偏低）
- 市场解读

示例内容：
```
比值,当前值,均值,标准差,分位数,状态,解读
金银比,61.34,83.89,9.09,4%,偏低 ⬇️,白银相对强势
金铂比,2.41,2.43,0.41,48%,正常 ➡️,均衡状态
...
```

#### 金银比_data_示例.csv
金银比的完整历史数据，包含：
- 时间戳
- 黄金收盘价
- 白银收盘价
- 金银比值

可用于：
- 进一步数据分析
- 导入Excel或其他工具
- 回测交易策略
- 建立历史数据库

## 如何使用示例

1. **查看图表**：直接打开PNG文件，了解图表样式和信息布局
2. **分析数据**：用Excel或Python打开CSV文件，进行自定义分析
3. **对比结果**：将你生成的文件与示例对比，验证工具运行正常

## 生成自己的文件

运行以下命令生成最新的分析文件：

```bash
# 生成综合分析
python analyze_all_ratios.py

# 生成金银比分析
python gold_silver_ratio_chart.py
```

生成的文件会保存在项目根目录，文件名包含时间戳。

## 注意事项

- 示例文件基于2026年2月9日的数据
- 实际运行时，数据会根据当前市场实时更新
- 不同时间生成的文件，数值和图表会有差异
- 这是正常现象，反映了市场的动态变化
